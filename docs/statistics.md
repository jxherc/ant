## Usage statistics collection in ant

By default, ant sends some statistics about how it's used. Currently, these include:

* Your operating system and computer language
* When you installed ant
* The version of ant that you're using
* How often you use certain features in ant
* An anonymous ID representing your installation of ant

Collecting this data allows us to prioritize which devices and operating systems we test and which features we work on.

ant does not send:
* Anything that can be used to personally identify you
* Any browsing history, passwords, or other data stored locally in ant

If you want to opt-out of statistics collection, you can do so by going to the preferences page within ant and de-selecting "send usage statistics".
